Albers Shuffle A screensaver-like JavaScript graphics program loosely inspired by some works of Anni Albers. The colors are picked randomly from a photo I took and change on each refresh of the page.
Bankgeheimnis A web app that helps you find good park benches to hang out on in the Rhein-Ruhr area in Germany. Since this is hosted for free on render.com, new visitors need to wait a minute or so for the server to start back up. Check out the GitHub page for more info.
